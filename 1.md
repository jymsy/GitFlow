GitFlow

安装
-----------

[下载git](https://git-scm.com/downloads)

配置
-----------
配置的是你个人的用户名称和电子邮件地址。这两条配置很重要，每次 Git 提交时都会引用这两条信息，说明是谁提交了更新，所以会随更新内容一起被永久纳入历史记录：

```sh
$ git config --global user.name "蒋羽萌"
$ git config --global user.email jiangyumeng@izaodao.com
```
